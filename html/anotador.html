What is it?

Añotador is a temporal tagger that annotates temporal expressions (DATEs, TIMEs, SETs, DURATIONs and optionally INTERVALs) in texts. It takes a text as an input and returns it as a JSON or a TimeML document. There are two different domains to choose: "General" (or Standard), that follows the TimeML standard, and "Legal", that includes specific domain considerations and tags, such as finding INTERVALS or normalizing differently some temporal expressions. This tool works for English and Spanish, altohught it is optimized for the Spanish language. An example of annotation can be found below:

<TIMEX3 tid="t1" type="DATE" value="2019-04-09">Today</TIMEX3> is <TIMEX3 tid="t2" type="DATE" value="2019-10-03">3 of October</TIMEX3>

We work <TIMEX3 tid="t1" type="SET" value="P1W" freq="02X">twice a week</TIMEX3>



How it works?

The scheme below illustrates how Añotador works.

First, we get as input the text and the anchor date (if none, we assume the current day) and we preprocess the text (lemmatization, POS tagging...). We then apply different types of rules in order to find the temporal expressions. Afterwards, we normalize the different types of expressions extracted by the rules. TO do so, we use as reference information from surounding expressions or the anchor date provided by the user. Finally, we return the text annotated in the requested format. .
